## D1. HR 대시보드 — `/dashboard` (hr_admin)

**진입:** hr_admin 로그인 기본 랜딩 / 사이드바 "대시보드".
**데이터:** `GET /dashboard/summary?cycleId=` → `{ data: { progress, distribution, unsubmitted, appeals, compensation } }` (요구사항 C-3).

## D2. 알림 — 상단 벨 드롭다운 + 알림 센터 `/notifications`

**목적:** 인앱 알림(읽음/안읽음). 일정 D-7/D-3/D-1·KPI 반려·결과 확정·이의제기 답변 등 트리거(요구사항 C-2). 상단 벨에 미읽음 뱃지, 드롭다운 미리보기, 전체 페이지에서 목록·필터·일괄 읽음.
**데이터:** `GET /notifications?unread=&type=&page=` · `PATCH /notifications/:id/read` · `PATCH /notifications/read-all`. 미읽음 카운트는 AppShell이 폴링/초기 fetch.

### D2-a. 상단 벨 드롭다운 (전 역할, AppShell 내)

기존 AppShell의 벨 버튼(이미 `notificationCount` 뱃지 렌더)을 **`NotificationBell`(신규)로 교체** — 클릭 시 `DropdownMenu`(shadcn) 패널 오픈.

```
        🔔③  ← 미읽음 3 (danger 점 뱃지, 기존 스타일 유지)
        └─ 클릭 ─────────────────────────────────────┐
           │ 알림                       [모두 읽음 ✓] │  헤더(헤어라인)
           ├──────────────────────────────────────────┤
           │ ● [⏰파랑] 본인평가 마감 D-3예요          │  unread = bg-primary/[0.04] + 좌측 점
           │           2026 정기평가 · 1시간 전        │
           │ ● [⛔빨강] KPI가 반려됐어요               │
           │           "가중치 합 확인" · 3시간 전     │
           │   [✅초록] 평가 결과가 확정됐어요          │  read = 평문
           │           2026 정기평가 · 어제            │
           ├──────────────────────────────────────────┤
           │              전체 보기 →                  │  → /notifications
           └──────────────────────────────────────────┘
           폭 360 · 최대 높이 420 scroll-area · 항목 클릭 → 읽음 처리 + 관련 화면 이동
```

타입별 아이콘/색(단일 액센트 원칙 — 시맨틱 색은 상태 기능으로 한정):
| type | 아이콘 | 톤(연배경+아이콘색, InfoBanner 팔레트 재사용) |
|------|--------|------|
| `deadline`(일정 D-7/3/1, 마감임박) | Clock | info `#EBF3FE`/`#1B64DA` |
| `kpi_rejected`(KPI 반려) | XCircle | warning `#FDECEC`/`#D6303D` |
| `result_confirmed`(결과 확정) | CheckCircle2 | success `#E7F8EF`/`#0F9457` |
| `appeal_answered`(이의제기 답변) | MessageSquare | tip `#FEF8EA`/`#C2670E` |
| `generic` | Bell | neutral `#F2F4F6`/`#4E5968` |

## D4. 엑셀 임포트/익스포트 (분산 배치 — 독립 화면 아님)

**목적:** xlsx 일괄 업로드(검증·오류 행 리포트) + 다운로드(결과·분포·보상) + 양식 다운로드(요구사항 C-1). hr_admin 권한.

- `/admin/audit`: 동일 패턴 감사 로그 내보내기.
