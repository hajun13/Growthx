# 레퍼런스 솔루션 — 화면 인벤토리 & UX 패턴 (참고용 / advisory)

`인사 평가 시스템 참고용/레퍼런스 솔루션 이미지/`의 타사 인사평가 SaaS 화면에서 추출한 화면 구조·컴포넌트·UX 패턴.

> ⚠️ **이 문서는 참고용(advisory)이다.** 화면 레이아웃·컴포넌트의 *아이디어 참고*로만 쓴다. **따라야 할 규칙·도메인·직책 체계는 PPT 운영계획 + [domain-model.md](domain-model.md) + [business-rules.md](business-rules.md)가 우선**이며, 충돌 시 그쪽이 이긴다. 레퍼런스 화면을 그대로 베끼지 말고 에너지엑스 요구사항·도메인에 맞게 재구성한다. 시각 언어는 루트 `DESIGN.md`(Kinetic Enterprise).
> 원본 스크린샷은 해당 폴더의 PNG를 Read. 데스크탑 웹 SaaS(넓은 레이아웃, 좌측 사이드바) 형태다.
> 🚨 **도메인 불일치 경고 — 가장 중요:** 이 타사 화면들의 **역량평가(S3 성과/공통역량/리더십역량 탭, S7 역량 평가 탭·레이더)·다면평가(S5 수평·상향 목록)·다면 비교(S7 본인/수평/1·2차)·S9 성과·역량 산점도(Scatter)** 는 **타사(advisory) 구조이며 우리 도메인이 아니다.** 에너지엑스 평가 = **KPI 성과(KpiScore)로 점수 산정 · 평가 유형 self(본인) + downward 3단계(팀장·본부장·그룹대표)** (수평·상향·다면 없음, 조직은 그룹→본부→팀→개인). 역량평가는 **참고용 백데이터로 별도 존재**(연봉·등급 0% 반영). 아래 화면 아이디어를 참고할 때 **수평/상향/다면 탭→채택 금지**, **다면 비교→self+downward 단계별 비교**, **레이더/산점도→KPI 분포·달성률 시각화**로 재구성하라(역량 탭은 참고용 조회 화면으로만). 본문은 레이아웃 아이디어로서 보존하되 위 매핑을 항상 적용한다.
> ⚠️ **점수·등급 예시 주의:** 아래 화면 설명의 점수/등급 숫자(예: B 79.5, A 82.5)는 **타사 제품 값**이며 우리 등급 척도와 다르다. 실제 등급은 [business-rules.md](business-rules.md) `gradeScale`(S 96~100, A 91~95…)로 산정한다. 레퍼런스 숫자를 그대로 쓰지 않는다.

## 핵심 화면

### S1. 인사평가 메인 (주차별 일정 캘린더)
- 일~토 7열 주간 캘린더. 각 평가 단계가 날짜 범위 바(진행중 파란 바)로 표시. (타사 예시 `평가준비 → 본인평가 → 다면평가 → 1차 부서장 → 2차 부서장`. **우리 도메인 흐름은 `평가준비 → 본인평가(self) → 1차 팀장(downward round1) → 2차 본부장(round2) → 최종 그룹대표(round3) → 결과·캘리브레이션` — 다면평가 단계 없음.**)
- 각 셀: 상태 배지(완료/미완료/N명 미완료) + 설명 + 액션 링크(평가하기/본인정보 확인하기/평가대상자 설정하기).
- 우상단: "인사평가 일정: [평가 진행용] 2024 하반기 인사평가", 공지사항.

### S2. 본인정보 확인 → 평가대상자 확인
- 본인 기본정보 카드 + **평가 프로세스 플로우**(본인평가 → 상향평가 → 수평평가 → 1차 부서장 평가 가로 화살표, 비공개 단계 흐림).
- 공통역량/리더십역량 항목 정의 목록(항목명 + 정의 + 평가문항).

### S3. 본인평가 작성 (성과 / 역량 탭)
> 🚨 **advisory only:** 아래 역량 탭(공통/리더십)·S~D 역량 라디오는 **타사 구조다.** 우리 본인평가 탭은 **KPI 2그룹(성과중심 / 협업·성장)** 이며 역량 탭은 없다. 항목은 KPI 과제(category·measureType별 달성률/건수)다. 3분할 레이아웃 아이디어만 참고.
좌-중-우 3분할:
- **좌:** 평가자 카드 + 차원 탭(`성과평가` / `공통역량` / `리더십역량`), 각 점수 요약.
- **중:** 평가 항목. 성과는 핵심전략·측정방식·가중치(100%)·성과 textarea. 역량은 항목별 정의 + **S/A/B/C/D 라디오** 선택, 항목 상태(미평가/완료).
- **우:** 과제 최종점수(예 82.50) 카드, 목표달성률(%), 실행기간(date~date), 가중치(%, 가중치 수정), 증빙서류 첨부(10MB 단일파일), 성과 리뷰 이력 조회.
- 상단 액션: `리뷰 불러오기`(성과관리 데이터 import), `증빙자료 관리`, `평가코멘트 보기`, `과제 추가`, `과제 삭제`.
- 우하단: `평가결과 제출`.

### S4. 분기 리뷰 모달
- 목표 유형 + 분기 탭(1분기/3분기/4분기). 분기별 "우수 작성"/"개선 작성" 코멘트 영역(평가 리뷰/셀프 리뷰).

### S5. 다면평가 목록
> 🚨 **advisory only — 우리 도메인 아님.** 다면(수평·상향) 평가 자체가 없다. 이 화면은 채택하지 않는다(부서장 평가 목록 S6로 대체). 목록+진행 도넛 레이아웃 아이디어만 참고 가치.
- 평가 현황(4명 중 0명 완료) + 진행 도넛.
- 표: 수평평가대상자 / 상향평가대상자 그룹. 컬럼: 성과·역량·공통역량·리더십역량 + `평가하기` 버튼 + 미완료 배지.
- 검색(이름/부서), `평가가이드 보기`, `평가결과 제출`.

### S6. 부서장 평가 (1차/2차) — 일괄 평가 + 분포
- 평가 그룹 탭(2차 부서장 평가 / 미배정 그룹).
- 진행 현황: 도넛(완료 2/2 100%) + 종합 합계/평균/편차 + **S/A/B/C/D 분포 막대 차트**.
- 표: 평가대상자별 종합·성과·역량·공통역량·직무역량 **점수 입력 셀** + `평가조회`. `결과 비교 조회`, `이전 평가 불러오기`, 페이지네이션(50개 보기).

### S7. 평가 상세결과 (다면 비교)
> 🚨 **advisory only:** 역량 평가 탭·레이더·수평평가 열은 **타사 구조다.** 우리 결과 비교는 **self(본인) + downward 3단계(팀장 round1·본부장 round2·그룹대표 round3)** (수평 없음, 역량은 참고용으로 별도 표시). 가로 막대 비교+전사평균 마커 레이아웃은 참고 가치.
- 탭: 종합 평가 / 성과 평가 / 역량 평가.
- 종합평가 배지(예 B 79.5) + 상위 %(percentile).
- 차원별 카드: 본인평가/수평평가/1차 부서장평가/2차 부서장평가 **가로 막대 비교**(각 등급+점수) + **전사 평균** 마커 + 타인 종합/타인 대비(A~D).
- **성과 평가 탭:** 과제별 가중치·종합·1·2차 부서장 점수 표 + 성과리뷰 코멘트(평가자 아바타 + 코멘트 말풍선).
- **역량 평가 탭:** 세부 역량 **레이더(오각형) 차트** — 본인 종합 vs 전사 평균 비교. 축: 공통역량·직무역량(공통)·직무역량(전문)·가치역량·서술형 피드백. + 역량별 본인/1·2차 부서장평가 막대 비교.

### S8. 관리자/설정 (사이드바 그룹)
- 평가 기본설정 / 인사평가 설정 / 인사평가 운영 / 평가결과 조회.
- **규칙 설정**(등급 구간·그룹 풀·인상률·가중치 정책 — business-rules의 RuleSet 편집), KPI 양식 관리(직급별), 평가 일정·대상자 설정, 전사 등급 분포 모니터링.

### S9. 평가 리포트 (조직 단위 분석) — `평가 리포트` 사이드바
- 헤더: `[결과 확인용] 2024년 상반기 인사평가` 주기 선택 드롭다운.
- **평가 등급 분포 막대 차트** (등급 분포 / 역량 분포 토글) — S/A/B/C/D 인원수.
- ~~**성과·역량 점수 산점도(Scatter)** — x: 성과 점수, y: 역량 점수~~ → 🚨 **advisory only:** 역량 축이 없으므로 산점도(성과×역량)는 채택하지 않는다. 대신 KPI 그룹별(성과중심×협업·성장) 점수 또는 달성률 분포 시각화로 재구성한다.
- **인사평가 결과 테이블** — 번호·이름·부서·직책(에너지엑스 체계: 대표이사/본부장/팀장/책임/선임/프로)·종합점수(등급+점수)·`리포트` 버튼. 검색.

### S10. 개인 평가리포트 (리포트 가이드) — S9의 `리포트` 진입
- 평가리포트 가이드 안내 배너.
- **평가 결과 카드 3종:** 종합평가(B 79.5)·성과평가(A 82.5)·백분위(B 75).
- **평가 단계별 평가자 흐름:** 본인평가 → 수평평가(비공개) → 1차 부서장 평가 → 2차 부서장 평가 (아바타 + 화살표).
- **이전 평가 결과 비교** (종합/성과/역량 토글) — 추이.
- AI 안내: 상단 `GPT 기술지원(Beta)` — 평가 리포트/코멘트 AI 보조(선택 기능, 향후).
